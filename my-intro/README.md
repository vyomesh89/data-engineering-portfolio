# My Portfolio Website

Personal portfolio website for **Vsk**, a Data Engineer and Cyber Forensics Expert.

## 🌐 Live Preview

Open `index.html` in the `my-intro` folder to preview the portfolio locally.

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Responsive design with modern visuals
- **JavaScript** — Interactive navigation and form behavior
- **Font Awesome** — Iconography
- **Google Fonts** — Typography (Inter)

## 💼 About This Project

This site showcases:
- Professional experience in data engineering and cyber forensics
- Key technical skills and certifications
- Featured project summaries
- Contact details with LinkedIn and GitHub links

## 🚀 Getting Started

```bash
cd my-intro
start index.html
```

## 📦 Deploy to GitHub Pages

This project is prepared to deploy from the `my-intro` folder using GitHub Actions. After pushing to GitHub, enable GitHub Pages in repository settings and select the Pages deployment workflow.

## 🔧 Customize

Update these files to personalize the portfolio:
- `index.html` — page content and profile sections
- `styles.css` — visual theme and layout
- `script.js` — animations and form behavior
